## Overview

Welcome to **Module 11: Advanced Python & Object-Oriented Programming**! This module builds on Python fundamentals and introduces you to the powerful paradigm of Object-Oriented Programming (OOP). You'll learn to design and implement classes, work with inheritance, understand polymorphism, and master advanced Python features that are essential for building scalable, maintainable applications.

## Learning Objectives

By the end of this module, you will be able to:

1. **Understand OOP Concepts**: Master the four pillars of OOP - encapsulation, inheritance, polymorphism, and abstraction
2. **Create Classes and Objects**: Design and implement custom classes with attributes and methods
3. **Work with Inheritance**: Build class hierarchies and understand inheritance patterns
4. **Implement Polymorphism**: Use method overriding and duck typing effectively
5. **Use Special Methods**: Implement magic/dunder methods for operator overloading
6. **Apply Decorators**: Use property decorators, class methods, and static methods
7. **Handle Exceptions**: Create custom exceptions for better error handling
8. **Use Context Managers**: Implement the context manager protocol for resource management
9. **Understand Composition**: Build complex objects through composition patterns
10. **Design Applications**: Apply OOP principles to real-world application design

## Prerequisites

- Completion of Module 10: Python Fundamentals
- Understanding of functions, data structures, and basic Python syntax
- Familiarity with error handling and file operations

## Module Structure

### Theory Sections (40-50 hours)

1. **Classes and Objects** (Section 01)
   - Introduction to OOP
   - Defining classes
   - Creating instances
   - Instance attributes and methods

2. **Instance vs Class Attributes** (Section 02)
   - Instance attributes
   - Class attributes
   - Understanding `self`
   - Attribute access patterns

3. **Methods Deep Dive** (Section 03)
   - Instance methods
   - Class methods (`@classmethod`)
   - Static methods (`@staticmethod`)
   - When to use each type

4. **Inheritance** (Section 04)
   - Single inheritance
   - Method overriding
   - Using `super()`
   - Multiple inheritance
   - Method Resolution Order (MRO)

5. **Polymorphism** (Section 05)
   - Method overriding
   - Duck typing
   - Abstract base classes
   - Operator overloading

6. **Encapsulation** (Section 06)
   - Public, protected, private attributes
   - Name mangling
   - Property decorators
   - Getters and setters

7. **Special Methods (Magic Methods)** (Section 07)
   - `__init__`, `__str__`, `__repr__`
   - `__len__`, `__getitem__`, `__setitem__`
   - Comparison operators (`__eq__`, `__lt__`, etc.)
   - Arithmetic operators (`__add__`, `__mul__`, etc.)
   - Context managers (`__enter__`, `__exit__`)

8. **Property Decorators** (Section 08)
   - `@property` decorator
   - Getter, setter, deleter methods
   - Read-only properties
   - Computed properties

9. **Abstract Base Classes** (Section 09)
   - ABC module
   - Abstract methods
   - Interface design
   - Enforcing contracts

10. **Composition vs Inheritance** (Section 10)
    - Composition pattern
    - "Has-a" vs "Is-a" relationships
    - Aggregation
    - Design principles
