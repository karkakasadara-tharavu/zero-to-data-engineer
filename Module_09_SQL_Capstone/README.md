# Module 09: SQL Capstone Project

## Overview
This capstone project integrates all skills learned throughout Modules 01-08. You will design, implement, and deploy a complete enterprise data solution including database design, ETL processes, data warehouse, and reporting.

## Prerequisites
- Completion of Modules 01-08
- SQL Server Developer Edition
- SSIS/Visual Studio
- Power BI Desktop
- 40+ hours for completion

## Project Overview

### Scenario: Global Retail Analytics Platform

You are the lead data engineer for **GlobalMart**, an international retail company. Your task is to build a complete data platform that:

1. **Consolidates** data from multiple source systems
2. **Transforms** data for analytical purposes
3. **Stores** data in an enterprise data warehouse
4. **Delivers** insights through interactive reports

### Business Requirements

- Executive dashboard showing key KPIs
- Sales analysis by region, product, time
- Customer segmentation and lifetime value
- Inventory optimization insights
- Supplier performance metrics

## Module Structure

### Phase 1: Design and Planning (Labs 76-77)
- Requirements analysis
- Logical data model design
- Physical database design
- ETL architecture planning

### Phase 2: Source Systems (Lab 78)
- OLTP database implementation
- Sample data generation
- Source system documentation

### Phase 3: Data Warehouse (Labs 79-80)
- Dimensional modeling
- Star schema implementation
- Slowly changing dimensions
- Historical tracking

### Phase 4: ETL Development (Labs 81-82)
- Extract from source systems
- Transform with business rules
- Load to data warehouse
- Error handling and logging

### Phase 5: Reporting (Lab 83)
- Power BI data model
- DAX measures library
- Interactive dashboards
- Deployment and security

### Phase 6: Final Presentation (Lab 84)
- Documentation
- Demonstration
- Code review
- Evaluation

## Evaluation Criteria

| Component | Points |
|-----------|--------|
| Database Design | 100 |
| ETL Development | 150 |
| Data Warehouse | 100 |
| Reporting | 100 |
| Documentation | 50 |
| **Total** | **500** |

## Deliverables

1. Database design documents
2. SQL scripts (all DDL and DML)
3. SSIS packages
4. Power BI reports
5. Technical documentation
6. Presentation slides
